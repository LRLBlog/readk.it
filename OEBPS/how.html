How does readk.it work?

Readk.it is implemented using HTML, CSS and JavaScript, the same technologies that are used with the eBook content.

Readk.it espouses Responsive Web Design techniques, which allow for suitable reading across a wide range of device sizes.

As readk.it functions as an integral part of the eBook and travels inside it, the EPUB file can be read conventionally on vendor reading systems (in which case, readk.it is effectively ignored).

Readk.it can be used to read the eBook content in a number of ways:

  • The EPUB can be served up to the reader by a web server, for example in the form of a subscription service
  • A web service can be provided which the reader can use to upload the EPUB file, and have it returned to them as web content that can be stored in local storage on their device; in this way, once they've initiated the title via the web service, they can then read the title on their device while offline.
  • The EPUB can be delivered as a web-based application in Firefox OS
  • A browser extenson can be used to unpack and load the EPUB file into the web browser
